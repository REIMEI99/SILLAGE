# SILLAGE / 《余香》 v0.12 — Canonical Rules

> v0.12 机械规则 source of truth。视觉仍以 B2.10 reference 为准。

## 1. 回合循环

1. 游戏开始时洗混 16 张顾客卡，并将两位顾客放到左右工作台，另有三位进入公开等候区。
2. 每回合从香料袋抽取 4 枚组成公共池；玩家最多进行 2 次投料，每次将公共池的一枚香料加入左瓶或右瓶。
3. 投料顺序是 FIFO，配方最多 6 枚。两次投料完成后，未使用的公共池香料回到袋中并重新洗混。
4. 回合末只为达到满意线的瓶子显示“交付 / 保留”选择；未达到满意线的瓶子静默保留。
5. 交付后顾客离开，空出的工作台从公开等候区补入顾客；没有可补顾客时静默进入下一回合。
6. 香料袋不足 4 枚时结束本局。

没有耐心、倒计时、自动离店或离店清瓶。顾客会一直等待，直到玩家交付。

## 2. 顾客卡

顾客卡固定为 16 张：

- 8 张单香气卡：每种香气各 1 张。
- 4 张双香气卡：柑橘 / 水生、绿意 / 果香、花香 / 琥珀、木质 / 草本。
- 2 张三香气卡：柑橘 / 绿意 / 花香，以及水生 / 果香 / 木质。
- 2 张特殊卡：主角明确、多面变化。

顾客类型只影响规则运算，不在 UI 中显示内部类型名。所有顾客的 `satisfactionLine` 都是 2，所有订单的 `orderScore` 都是 3。

## 3. 满意度

配方长度必须为 3–6 枚，且满意度达到 +2 才能交付。满意度只决定能否交付，不计入得分。

### 单香气

- 目标香气：每枚 +2。
- 环上距离 1–2 的香气：0。
- 环上距离 3–4 的香气：−1。

### 双香气

- 两种偏好香气：每枚 +1。
- 两端外侧相邻香气：0。
- 其余香气：每枚 −1。

### 三香气

- 三种偏好香气：每枚 +1。
- 其余香气：0。

### 特殊卡

- 主角明确：`最高香气重复次数 − 1`。例如 AAA 的满意度为 +2。
- 多面变化：`不同香气种类数 − 1`。例如 ABC 的满意度为 +2。

特殊卡不使用逐枚香气加减分。

## 4. 得分

每次成功交付获得固定订单分 3 分，再加一项最高技法分：

- 浓烈：0–2 分；
- 纯粹：0–2 分；
- 层叠：0–2 分；
- 完美技法：技法分 2 分 + 完美奖励 6 分，共 8 分技法分。

同一瓶最多结算一种技法。顾客满意度、偏好距离和特殊结构只影响交付资格，不直接加到最终分数。

## 5. 规则函数边界

所有香气权重、满意度、可交付判定、技法和预演都必须由 pure functions 计算。React UI 只读取结果，不硬编码规则结果。
