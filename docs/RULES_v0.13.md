# SILLAGE / 《余香》 v0.13 — Canonical Rules

> v0.13 机械规则 source of truth。视觉仍以 B2.10 reference 为准。

## 核心回合

游戏固定进行 20 回合。每回合从香料袋抽取 4 枚组成公共池，玩家执行 2 次投料；每次把一枚公共池香料加入左瓶或右瓶。未使用的 2 枚回袋并重新洗混。瓶子使用 6 格 FIFO。

投料后，只为满足交付条件的瓶子显示“交付 / 保留”；其他瓶子静默保留。所有可交付瓶子完成选择后自动继续。第 20 回合完成交付结算后立即结束，不再补入新顾客。

顾客没有耐心、倒计时、自动离店或离店清瓶。

## 顾客与满意度

配方必须有 3–6 枚，且满意度至少 +2 才可交付。满意度只决定能否交付，不计入最终得分。

- 单香卡：目标香气每枚 +2；环距 1–2 为 0；环距 3–4 为 −1。
- 双香卡：两种偏好每枚 +1；卡面指定的一种对立香气每枚 −1；其余为 0。
- 三香卡：三种偏好每枚 +1；其余为 0。
- 主角明确：最高香气重复次数 − 1。
- 多面变化：不同香气种类数 − 1。

所有顾客满意线为 +2，订单固定 3 分。

## 技法与得分

每瓶只结算最终分数最高的一种技法。普通技法仍为 0 / 1 / 2 分；完美技法包含 2 分技法分和按路线区分的完美奖励：

- 完美浓烈（AAAAB 家族）：完美奖励 +6，技法合计 8 分，订单总分 11。
- 完美纯粹（AAABB / AAABBB 家族）：完美奖励 +5，技法合计 7 分，订单总分 10。
- 完美层叠（ABCDEF）：完美奖励 +5，技法合计 7 分，订单总分 10。

结构重叠时仍只取最高结果。例如 AAAABB 同时符合浓烈与纯粹，但浓烈总分更高，因此按浓烈结算。

## 数据与规则边界

规则版本写入每局记录。香气权重、满意度、可交付判定、技法、得分和公共池预演均由 `src/rules/` 中的 pure functions 计算，UI 不硬编码结果。
