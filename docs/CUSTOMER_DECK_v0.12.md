# SILLAGE v0.12 — 顾客卡池

v0.12 使用 16 张顾客卡。每位顾客都要求满意度至少 +2，订单固定 3 分，配方长度必须为 3–6 枚。

| 数量 | 卡组 | 偏好 / 结构 |
| ---: | --- | --- |
| 8 | 单香气 | 柑橘、水生、绿意、果香、花香、琥珀、木质、草本各 1 张 |
| 4 | 双香气 | 柑橘/水生、绿意/果香、花香/琥珀、木质/草本 |
| 2 | 三香气 | 柑橘/绿意/花香；水生/果香/木质 |
| 2 | 特殊 | 主角明确；多面变化 |

双香气卡的偏好香气每枚 +1，两端外侧相邻香气为 0，其余为 −1。三香气卡的三种偏好香气每枚 +1，其余为 0。单香气卡的目标香气每枚 +2，环上距离 1–2 为 0，距离 3–4 为 −1。

特殊卡不使用逐枚权重：

- **主角明确**：最高香气重复次数 − 1，AAA 达到 +2。
- **多面变化**：不同香气种类数 − 1，ABC 达到 +2。

顾客类型是内部数据字段，不作为 UI 标签展示。所有正权重香气都应在中心香气环上保持高亮；特殊卡没有逐枚正权重，因此不高亮具体香气。
