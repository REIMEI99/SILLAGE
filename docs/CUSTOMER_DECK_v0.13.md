# SILLAGE v0.13 — 顾客卡池

卡池仍为 16 张：8 张单香、4 张双香、2 张三香、2 张特殊卡。所有顾客满意线为 +2，订单固定 3 分。

## 双香卡

双香卡承担连接集中构筑与层叠构筑的职责。每张卡只有一种明确对立香气：

| 顾客 | 偏好香气（每枚 +1） | 对立香气（每枚 −1） | 其余 |
| --- | --- | --- | --- |
| 晨跑后推门进来的编辑 | 柑橘 / 水生 | 花香 | 0 |
| 刚从山里回来的人 | 绿意 / 果香 | 木质 | 0 |
| 夜班结束后还不想回家的人 | 花香 / 琥珀 | 柑橘 | 0 |
| 刚从排练室出来的舞者 | 木质 / 草本 | 绿意 | 0 |

对立香气采用统一生成原则：取第一偏好在八香气环上的正对香气，并显式保存到顾客 JSON，UI 与规则函数都读取该字段。

这个结构允许双香顾客在避开对立香气时接纳完美层叠，同时保留一个明确的回避点。

## 其他顾客

- 单香卡继续专注浓烈 / 纯粹路线，不保证兼容完美层叠。
- 三香卡继续作为接住半成品的宽容顾客。
- 主角明确继续支持集中与重复结构。
- 多面变化继续支持多种香气与层叠结构。

内部类型名不显示在 UI；所有正权重香气在中心香气环高亮，对立香气在顾客卡上明确标注。
